====  Kodi plugin for simple channel list ====

Installation: 
  A) Copy this folder into .kodi/addons/. and refresh addon list 
    I.e.
      cd $HOME/.kodi/addons
      curl -L https://github.com/YggdrasiI/plugin.video.deutsche_sender/archive/master.zip > plugin.video.deutsche_sender.zip
      unzip plugin.video.deutsche_sender.zip
      kodi-send -a "UpdateLocalAddons"

  B) Checkout this repository and call 'make' to create an addon-zip.
     Intall the zip over the Kodi GUI.

  Note for  Kodi 17.0 (Krypton) and above:
      You had to allow addons from unknown sources in Preferences>System>Addons )
  
	

Tested with:
  Kodi 16.1 / LibreELEC 7.0.2
  Kodi 17.1 / LibreELEC 8.0.1
