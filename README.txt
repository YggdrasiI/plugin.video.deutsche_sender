====  Kodi plugin for simple channel list ====

Installation: 
  Copy plugin into .kodi/addons/. and refresh add-on list (or restart Kodi)
  
  I.e.
    cd $HOME/.kodi/addons
    curl -L https://github.com/YggdrasiI/plugin.video.deutsche_sender/archive/master.zip > plugin.video.deutsche_sender.zip
    unzip plugin.video.deutsche_sender.zip
    kodi-send -a "UpdateLocalAddons"
	

Tested with:
  Kodi 16.1 / LibreELEC 7.0.2
